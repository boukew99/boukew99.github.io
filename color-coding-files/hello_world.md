# Hello World in different languages. `Hello World`

## C

```c
#include <iostream>
 
int main() {
    std::cout << "Hello World";
    return 0;
}
```

## C++

```cpp
#include <iostream>
 
int main() {
    std::cout << "Hello World";
    return 0;
}
```

## Matlab

```matlab
disp('Hello World');
```

## C#

```csharp
namespace HelloWorld
{
    class Hello {        
        static void Main(string[] args)
        {
            System.Console.WriteLine("Hello World");
        }
    }
}

```

## Haskell

```haskell
module Main where

main :: IO ()
main = putStrLn "Hello World"
```

## Pascal

```pascal
program Hello;
begin
  writeln ('Hello, world.');
end.
```
