Projects review

I have done many programming projects, but so many end up being just one thing. Then it is done, never to be opened again, just a learning experience. Some projects have been published or preserved or screenshotted. Might be fun to look back and see what I learned from them and now.

Hello Python

My first connection with programming. I mean remember being intoxicated by all the possible combinations you could make with pytons syntax and thinking I never would have to solve something manually.

Hello GameMaker

Before I was already interested in game logic/design and the graphical nature of it, so I tried to learn and make games in GameMaker, but I never really got into it.

Hello Godot

Then I met Godot and though being somewhat strange it had a certain simplicity and openess, which made me interested in the technologies behind it. I read all the blogs dilligently, not understanding much of the technical terms, but still learning. I ended up with learning a lot of the different approaches to programming, but ulitmately honed the code simplicity where leveraged almost all behavior with the provided nodes. I particulary liked desiging minimal User Interfaces with the container class. This produced some opensourced addons and demos. There are also some 2D games for game jams lying around.

Scratch Animator

I got interested in animation techniques, effective art styles, production and supportive technology. The productions of Studio Ghibli have many details integrated, but are enormosly intensive to make. With ponyo they used a simplfied art style to give more detail to the animations instead of the image, but the reception was mixed. How can you produce a pure animative experience without comprimising on visual image needs, while also being condensely packed? Looom got an interesting approach where they only support weave animations. Their app seemed approachble from a usability standpoint and people where happy with it, so I thought to create my own specific tool and workflow with the new found knowledge of User interface Nodes in Godot.
