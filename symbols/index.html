Pixel Icons & Symbols

Icons of resolution of 16x16 px mostly only one color, which makes it binary and modulatable. The difference between icon and symbols?. A symbol is an familiar instance of an abstract concept. It is licensed X.

Usage

  1. Look for a suitable icon
  2. Save Image As... on the image by right-clicking on it.
  3. Optionally Repeat from 1
